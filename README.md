# infoservice_server

## Project setup
```
composer install
```
<p>Copy .env.example file to .env</p>
<p>Set data for database (DB_*)</p>
<p>Set data for mail (MAIL_*)</p>

```
php artisan migrate
```

## Project start
```
php artisan serve
```
